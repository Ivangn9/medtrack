# Logos de Marcas

Colocá acá las imágenes de logos con el nombre normalizado de la marca.

## Regla de nombres
El nombre del archivo debe ser el nombre de la marca en **minúsculas**, con espacios y caracteres especiales reemplazados por guiones (`-`), sin tildes.

## Ejemplos
| Marca en el sistema | Nombre de archivo |
|---|---|
| Siemens | `siemens.png` |
| GE Healthcare | `ge-healthcare.png` |
| GE | `ge.png` |
| Philips | `philips.png` |
| Canon | `canon.png` |
| Agfa | `agfa.png` |
| Carestream | `carestream.png` |
| Planmeca | `planmeca.png` |
| Bayer | `bayer.png` |

## Formatos soportados
- `.png` (recomendado, con fondo transparente)
- `.jpg` / `.jpeg`

## Cómo subir
1. Ir a `github.com/Ivangn9/medtrack/logos`
2. "Add file" → "Upload files"
3. Subir el archivo con el nombre correcto
4. El logo aparece automáticamente en todos los equipos con esa marca
